# ai-data-engineering
practical data engineering for AI application

## Multimodel data pipeline
End-to-end pipeline for semantic search over meeting recordings and slides. 
Designed and built using AI-assisted development, with full evaluation including 
WER comparison across Whisper models, Precision@5 retrieval scoring, and chunk 
size ablation.  
**Tech:** Python · Whisper · Tesseract · sentence-transformers · ChromaDB
